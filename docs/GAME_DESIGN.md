# ib-lol talk — Game Design Document

## 개요
리그 오브 레전드 라인전을 오마주한 웹 브라우저 기반 Interactive Fiction 게임.
텍스트 서술과 이미지로 라인전 1v1 대결을 진행한다.

## 핵심 규칙

### 플랫폼
- 웹 브라우저 (HTML/CSS/JS)

### 게임 모드
- 1v1 라인전
- 초기: AI 상대 (싱글플레이)
- 추후: 멀티플레이어 지원

### 턴 구조: 동시 선택
- 매 상황마다 양쪽이 동시에 행동을 선택
- 양쪽 선택의 **조합**이 결과를 결정 (확률 없음, 순수 전략)
- 선택지는 자연어로 표현되며, 행동의 의도가 명확히 포함됨

### 승리 조건 (먼저 달성한 쪽 승리)
1. **킬** — 상대 HP를 0으로 만듦
2. **CS 100** — 미니언 100마리 먼저 처치
3. **타워 파괴** — 상대 타워 체력을 0으로 만듦

### 제약
- 정글러 없음 (순수 1v1)
- 귀환 1회 가능 (아이템 구매)

## 챔피언

### 초기 지원: 리신 (Lee Sin)
추후 챔피언 추가 예정.

#### 기본 스탯 (레벨 1, 실제 롤 수치)
- HP: 645
- 기력(Energy): 200
- AD: 68
- 방어력: 36
- 마법저항력: 32.1
- 공격속도: 0.651
- 이동속도: 345

#### 스킬
- **패시브 (연타)**: 스킬 사용 후 2회 기본공격 공속 40% 증가, 기력 회복 (첫 타 15, 둘째 타 15)
- **Q (음파/공명타)**: 음파 발사 → 적중 시 재사용으로 돌진. 물리 피해.
- **W (방호/철갑)**: 대상에게 돌진 + 쉴드. 재사용 시 생명력 흡수.
- **E (폭풍/쇠약)**: 범위 마법 피해. 재사용 시 둔화.
- **R (용의 분노)**: 대상 넉백 + 물리 피해.

#### 스킬 수치 (레벨 1 기준)
- **Q1 (음파)**: 55 물리 피해 + 추가AD 115% | 기력 50 | 쿨 10초
- **Q2 (공명타)**: 55 물리 피해 + 대상 잃은 체력 비례 추가 피해 | 기력 30
- **W1 (방호)**: 50 쉴드 (2초) | 기력 50 | 쿨 12초
- **W2 (철갑)**: 생명력 흡수 5% (5초) | 기력 30
- **E1 (폭풍)**: 35 마법 피해 + 추가AD 100% | 기력 50 | 쿨 9초
- **E2 (쇠약)**: 20% 둔화 (4초) | 기력 30
- **R (용의 분노)**: 175 물리 피해 + 추가AD 200% | 쿨 90초

## 맵 / 위치 시스템

### 1차원 레인 위치 (단순화)
```
[아군타워] --- [아군쪽] --- [중앙] --- [적쪽] --- [적타워]
```

### 추가 위치
- 윗 부쉬 / 아래 부쉬 (중앙 양 옆)

### 거리 개념
- 인접한 구역: 근접 (기본공격, E 가능)
- 1칸 거리: 중거리 (Q 가능)
- 2칸 이상: 원거리 (스킬 불가)

## 선택지 시스템

### 하이브리드 방식
1. 게임 엔진이 현재 상태 기반으로 가능한 행동 목록 생성
2. 각 행동을 자연어 템플릿으로 변환하여 선택지 표시
3. 양쪽 선택 조합으로 결과 결정 (확률 없음)

### 선택지 생성 조건
- 현재 위치, 거리
- HP / 기력 / 쿨타임
- 미니언 웨이브 상태
- 레벨, 스킬 레벨
- 소환사 주문 상태

### 결과 결정: 조합표
양쪽의 선택을 조합하여 결과가 결정됨.

예시:
| 내 선택 | 적 선택 | 결과 |
|---------|---------|------|
| Q 발사 | 제자리 CS | Q 적중 → 피해 |
| Q 발사 | 옆으로 이동 | Q 빗나감 |
| Q 발사 | 앞으로 돌진 | Q 적중 + 근접 상황 |
| CS 정리 | CS 정리 | 양쪽 CS 증가, 평화로운 턴 |
| 부쉬 진입 | Q 발사 | Q 빗나감 (시야 밖) |

### 행동 카테고리
- **공격**: Q/W/E/R 사용, 기본공격
- **CS**: 미니언 막타
- **이동**: 앞/뒤/부쉬/강가
- **방어**: 쉴드, 회피 기동
- **전략**: 귀환, 소환사 주문

## 미니언 시스템
- 일정 턴(상황) 간격으로 웨이브 도착
- 미니언은 자동으로 서로 교전
- 플레이어가 막타를 쳐야 CS로 인정
- 미니언이 많은 쪽이 라인 주도권 유리

## 레벨 시스템
- 미니언 처치 및 챔피언 교전으로 경험치 획득
- 레벨업 시 스킬 포인트 배분
- 실제 롤 레벨업 경험치 테이블 참고

## 아이템 시스템
- 귀환 1회 시 골드로 아이템 구매
- 시작 아이템: 도란의 검 + 체력 포션 1개
- 구매 가능 아이템: 추후 정의 (리신 추천 빌드 기반)

## 소환사 주문
- 점멸 (Flash): 즉시 1칸 이동 (쿨 300초)
- 점화 (Ignite): 대상에게 지속 피해 (쿨 180초)
- 순간이동 (Teleport)
- 탈진 (Exhaust)
- 등 — 게임 시작 전 2개 선택

## 룬 시스템
- 게임 시작 전 룬 선택
- 초기에는 대표 룬 세트 몇 가지 프리셋 제공
- 추후 상세 커스터마이징

## UI 구성 (웹)
- 상단: 상황 서술 텍스트 + 이미지
- 중단: 양쪽 상태바 (HP, 기력, 레벨, CS)
- 하단: 선택지 버튼들
- 사이드: 아이템, 스킬 쿨타임 정보
