## 피해 계산 공식 (롤 기본)

### 물리 피해 감소
```
실제 피해 = 물리 피해 × 100 / (100 + 방어력)
```
방어력이 음수일 경우:
```
실제 피해 = 물리 피해 × (2 - 100 / (100 - 방어력))
```

### 마법 피해 감소
```
실제 피해 = 마법 피해 × 100 / (100 + 마법저항력)
```

### 예시: 레벨 1 리신 vs 리신
- Q1 피해: 55 + (115% × 0) = 55 (추가AD 없으므로 기본 피해만)
  - 실제 피해: 55 × 100 / (100 + 36) = 55 × 0.735 = **40.4**
- E1 피해: 35 + (100% × 69) = 104 (마법 피해, 총AD 적용)
  - 실제 피해: 104 × 100 / (100 + 32) = 104 × 0.758 = **78.8**
- AA 피해: 69
  - 실제 피해: 69 × 100 / (100 + 36) = 69 × 0.735 = **50.7**
- R 피해 (레벨 6): 175 + (200% × 0) = 175
  - 실제 피해: 175 × 0.735 = **128.6**

> 참고: 도란의 검 (+10 AD) 착용 시 추가AD = 10

### 도란의 검 착용 시 레벨 1
- 총 AD: 79, 추가 AD: 10
- Q1: 55 + (1.15 × 10) = 66.5 → 실제: **48.9**
- E1: 35 + (1.0 × 79) = 114 → 실제: **86.4**
- AA: 79 → 실제: **58.1**
- HP: 645 + 80(도란) = 725 (도란의 검 체력 보너스 없음, 패치에 따라 다름)
